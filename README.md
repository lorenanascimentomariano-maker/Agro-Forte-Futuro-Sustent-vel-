# Agro-Forte-Futuro-Sustent-vel-
Agro Forte, Futuro Sustentável

Sobre o Projeto

Este site foi desenvolvido como parte das atividades da Alura com o tema “Agro Forte, Futuro Sustentável”.

O objetivo é apresentar a importância do agronegócio para a sociedade e destacar práticas sustentáveis que ajudam a preservar o meio ambiente para as futuras gerações.

Conteúdos Abordados

* O que é o agronegócio;
* Sustentabilidade no campo;
* Tecnologias sustentáveis;
* Agricultura de precisão;
* Energia renovável no meio rural;
* Importância da preservação ambiental.

Tecnologias Utilizadas

* HTML5
* CSS3

Objetivo

Demonstrar como a tecnologia e a sustentabilidade podem trabalhar juntas para fortalecer a produção agrícola, garantindo desenvolvimento econômico e preservação dos recursos naturais.
